# Parcial 1 WEB DEV ISIS-3710 - Node-Express-React

## Requirements
* Node.js

## Installation
Prepare the data

Go to data folder
`cd data`
Import the data
`mongoimport -d dbname -c collectionname --jsonArray --file file`

In the applicaotion main folder
`yarn install` or `npm install`

In the front folder
`yarn install` or `npm install`
